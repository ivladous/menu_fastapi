# Restaurant menu REST API

**Menus, submenus and dishes**

*Python 3.10+*

*SQLite*

## Install

```commandline
python -m venv venv
```
```commandline
. ./venv/Scripts/activate
```
```commandline
pip install -r requirements.txt
```
```commandline
alembic upgrade head
```
```commandline
uvicorn main:app
```
